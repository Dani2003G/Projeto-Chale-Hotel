## 📁 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/Dani2003G/Projeto_Chale_Hotel

    # Entrar no diretório
    $ cd Projeto_Chale_Hotel

    # Instalar as dependências
    $ yarn install

    # Iniciar o projeto
    $ yarn start

```
